How It Works

Four steps.
Full context, preserved.

Click each step to see a live preview of what happens.

01

Chat normally on any LLM

Use Claude, ChatGPT, or Gemini as normal. Build context over a long session.

02

Click the extension icon

When near the token limit, click the toolbar icon to open the popup.

03

Markdownify & Download

Extension captures DOM → sends to local FastAPI → returns clean Markdown → auto-downloads.

04

Continue in any new LLM

Upload context.md to a new session and pick up exactly where you left off.

Tech Stack

Built lean.
No bloat, no cloud.

A small, purposeful stack. Browser extension talks to a local Python backend. That's it.

Frontend
Manifest V3

Chrome Extension

Modern Manifest V3 with a service worker. Compatible with Chrome, Edge, and Brave.

Frontend
popup.js · content.js

Vanilla JavaScript

Zero dependencies on the browser side. Lightweight scripts that read the live DOM.

Backend
FastAPI · Python 3.8+

Local FastAPI Server

Runs on localhost:5000. Receives raw HTML, returns clean Markdown. Private by design.

Converter
html2text · markdownify

HTML → Markdown

Strips all UI noise and extracts the pure conversation into a portable .md file.

STEP 01

Clone the repo & start the backend

Run these commands in your terminal. Keep the server running while using the extension.

Terminal
# 1. Clone the repository
$ git clone github.com/Aritra-Karan47/llm-context-extension
$ cd llm-context-extension/backend
# 2. Install Python dependencies
$ pip install -r requirements.txt
# 3. Start the server — keep this running!
$ python app.py

STEP 03

You're set up — here's how to use it

Whenever you hit the context limit, follow these three actions.

1
Chat on any LLM until you see the context warning
Go to Claude, ChatGPT, or Gemini and have your conversation normally.
2
Click the ⟳ icon → "Markdownify & Download"
A context.md file downloads automatically.
